## One-Line Description
A student wellness web app that helps users track mood, sleep, hydration, exercise, stress, and daily self-care habits while giving simple wellness insights and safe support resources.

## Tech Stack
- React + TypeScript + Vite
- Tailwind CSS
- React Router
- Recharts
- Lucide React
- LocalStorage (no backend, no auth)

## Features
1. Daily wellness check-in (mood, stress, sleep, water, exercise, notes)
2. Dashboard with wellness score, charts, and suggestions
3. Guided breathing exercise (4-4-6 technique)
4. Wellness history with search and sort
5. Support resources and wellness tips with emergency disclaimers

## Local Development
```bash
npm install
npm run dev
```

## Scripts
- `npm run dev` — start dev server
- `npm run build` — production build
- `npm run preview` — preview production build
- `npm run lint` — ESLint
- `npm run test` — Vitest unit tests
